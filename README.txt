# GitHub Pages جاهز

## النشر السريع
1. اعمل حساب أو سجّل دخولك على GitHub.
2. أنشئ Repository جديد.
3. ارفع كل الملفات الموجودة في هذا المجلد كما هي.
4. ادخل إلى:
   Settings > Pages
5. عند Source اختر:
   Deploy from a branch
6. اختر:
   Branch: main
   Folder: / (root)
7. اضغط Save.
8. انتظر دقيقة أو دقيقتين ثم ستحصل على رابط مثل:
   https://USERNAME.github.io/REPO-NAME/

## على iPhone
- افتح الرابط في Safari
- اضغط Share
- اختر Add to Home Screen

## الملفات
- index.html : التطبيق الرئيسي
- manifest.json : إعدادات الويب آب
- sw.js : تشغيل بسيط أوفلاين
- icon-192.png / icon-512.png : أيقونات التطبيق
